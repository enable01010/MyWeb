# プログラマーとは
【パソコンにどんな動きをして欲しいか】を作るお仕事です。<br>
そして、プログラマーがPCにそれを伝える言葉がプログラムになります。<br>

# C言語とは
プログラムには色々な言語があります。<br>
処理が早くて容量が軽い車などに使われるC<br>
処理が早くて、複雑なものが作りやすいゲーム制作に使われるC++<br>
C++を簡単に使えるようにした代わりに処理が重たくなったC#<br>
AI開発などで使われるPython<br>
サーバー処理で使われるJava<br>
など日本語、英語、ドイツ語みたいにいろんな言語が分野によって使い分けられています。<br>
<br>

## プログラムの解説 

### void main()
プログラムにはエントリーポイントと言われるプログラムの開始地点があります。<br>
今回はこのvoid main()と書いてある直後の行を上から順番に実行していく設定になっています。<br>
<br>
このvoid main()の波括弧　｛　｝　の間に書いてあることをPCが1行ずつ順番に実行していきます。<br>

## Consle.WriteLine("自分の名前");
プログラムには関数と言われるものがあります。<br>
関数とはPCにやらせる動作のことです。<br>
この Consle.WriteLineは【文字を表示する】という動作です。<br>
""の中身を変えると表示される名前が変わります。<br>
このPCにやらせる動作は他にもいろいろあります。<br>
・計算をしろ。<br>
・画像を表示しろ。<br>
・キーボートの入力を受け取れ。<br>
・データを保存しろ。<br>
このような指示を組み合わせることで世の中のアプリやゲームは動いています。<br>
例えば、<br>
1行目　キーボードのDキーを受け取れ。<br>
2行目　キャラクターが右に１進んだ位置を計算しろ。<br>
3行目　キャラクター画像をその位置に表示しろ。<br>
みたいな風にプログラムを書くことでDキーを押すことで右側に進めるゲームの処理ができます。<br>

# 入力を受け取ってみよう
## 文字の入力を受け取るプログラムを書いてみよう
プログラムが書けたらさっきと同じ方法で起動してみましょう。<br>

~~~ clike

void main()
{
	Console.ReadLine();
}

~~~

## 動作チェック
画像のように自由に文字を入力できると思います。<br>
そしてEnterキーを押したら入力が完了すると思います。<br>

## プログラムの解説
### Console.ReadLine();
この関数は、【文字を読み取れ】という関数です。<br>
Console.WriteLine()で表示しろ（Wirte　書け、　Line　1行）<br>
Console.ReadLine()で読みてれ（Read　読め、　Line 1行）<br>
となっています。<br>
Paizaなどでよく使うため必ず覚えましょう。<br>

# 入力した文字を活用してみよう
今のままだとただ入力を受け取っただけで、何も出来ないと思います。<br>
なので、この入力をデータとして活用してみましょう。<br>

## 入力を受け取るプログラムを書いてみよう
プログラムが書けたらさっきと同じ方法で起動してみましょう。<br>

~~~ clike

void main()
{
	string line = Console.ReadLine();
	Console.ReadLine("あなたが入力したのは");
	Console.ReadLine(line);
}

~~~
